# リモートワーク
転職してリモートワークとなり、約半年ほどになる。  
朝の通勤時間や昼休みが自分の空間で好きに過ごせるのはありがたい。私はその時間で本を読んだり、メモやこの日記を書いたり、猫と戯れたりしている。我が家の愛猫 ぽんちょもリズムに慣れてきたのか、最近は昼休みになると「ニャー」と誘ってくることが多い。うれしい。  
一方で、子どもが学校から帰宅する夕方〜夜の時間は課題を感じている。所属する会社のリズム的に、夕方〜夜の時間がコアタイムなことが多く、向こうの部屋でワイワイだったり、モゾモゾだったり、何か物語が進んでそうだけれど、なかなか参加できない。近くにいるのに入っていけないもどかしさのようなものがある。  
子どもの記憶的にも「父ちゃんは家にはいたけど、別部屋に篭って…」という感じだろうか。近くにいるのに惜しい。  
これはリモートワークと時間配分・スケジューリングの自由を混同していたのだろう。リモートワークだからといって、仕事とプライベートの時間を自由に配置できるわけではない。  
課題を感じているとはいえ、仕事の合間に子どもの音読を聞いたりできるのはうれしい。これまでの仕事のあり方は、家庭のことを大きな部分諦めていたのだと気づく。諦めていたというか、存在にも気づいていなかった部分も大きいかもしれない。一方で私が意識を向けられる量は有限に違いない。家庭に意識を向けた分、仕事なのか、他の何かが薄まっている可能性はありそう。  
仕事とプライベートのバランスを、タイミングごとに上手く切り替えながら毎日生きていく…これはなかなか高度なスキルが必要だろう。  
そんな模索が今日も続く。
