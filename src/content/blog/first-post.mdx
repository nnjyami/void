# あたらしい日記
新年を迎え、装い新たに、真新しいフレームワークを使って、久しぶりに web で遊んでみる。昔、必至になって Wordpress をいじくりまわしていた時を思い出しつつ、Astro で組んでいる。
最近、読書がすっかり習慣化して、ほぼ毎日、何かしら本を読んでいる。その影響もあってか、ものすごく縦書きで表現したい。
日記を書くだけなら note とかで書いたらいい。でも、なぜか見た目の表現にもこだわってしまう悪い癖。どうせなら、文庫本や気品のある本のように、美しい明朝体が使いたくて、筑紫オールド明朝にしてみた。ひらがなの流れるようなはらいが美しい。本当は漢字とひらがなの大きさ、字間とかもう少し調整できると良いのだけれど、そこは追々調整だろうか。
このフォーマットをつくる上で、新聞の構成も参考にした。久しぶりに新聞を買って読んでみた。限られた面積の中で重要なものをパッと一目で伝えつつ、ボリュームのあるテキストを流していく新聞の構成は面白い。
使用できる面積は無限大な故に、間延びしている web も参考にするべきだ。web に合わせた新聞のような圧縮レイアウトは何だか可能性を感じる。
昨年暮れあたりから、あたらしい日記の構想があり、頭の中でぐるぐる回していて、今週末ついにつくり始めた。Webフォントを使い、縦書きの調整とかしていたら、思いの外、知らない設定があり、なかなか苦戦した。そして実機で見ると何故かフォントサイズが全然違う…。何事も手を動かさないとわからないことは多い。そして、つまづいても、何かが出来上がることはとても楽しい。
毎日何かしら書くことで、新たな発見を積めたら良いなぁと思う。
