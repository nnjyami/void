# 生成AI
先ほどから、何を書こうか、真っ白なメモ帳を眺めている。そうやって悩むときは何も思い浮かんでこない。  
実際、掘り起こしてみると、ああいうこともこういうこともあるに違いないのに。  
そんな時、メモ帳の UI 的に何かできることはあるだろうか。    
白いキャンバスを眺めていると、今週あったキーワードや考えたかったことがぼんやり浮かぶ。  
GitHub の copilot とつながった VSCode で文章を書こうとすると、近い体験ができる。こちらは書こうとすると、「あんたの続きはこれでしょ」という感じで文章が薄く出てくる。コードの場合はこれがうれしく、気持ちいいが、文章だと違うと思ってしまう。
今年の芥川賞を受賞した九段理江さんは ChatGPT を使いながら書いているらしい。どうして文章のプロが使っていて、素人の自分が使わないのか。  
自分でやりたいと思ってしまう場面は、対AI以外にもありそうだ。この時の感情は何なんだろうか？所有欲？自己顕示欲？  
つくるプロセスは大切で、その過程が一番楽しかったりもする。AIを使うと楽しさが楽しさが奪われると思っているのだろうか。  
凝り固まった思考や欲は投げ捨てて、軽やかに生きる方が楽しそうにも思える。楽しくAIと生きる方法を考えたい。
</VBlock>
