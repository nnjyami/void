# 生成AI
先ほどから、何を書こうか、真っ白なメモ帳を眺めている。そうやって悩むと、何も思い浮かんでこない。  
実際は、あれもこれも何かしらあるに違いないのに。  
そんな悩める自分に、メモ帳ができることはあるだろうか。白いキャンバスを一定時間眺めていると、今週あったキーワードや考えたかったことがぼんやり浮かぶとかどうだろう。  
GitHub Copilot とつながったVSCodeで文章を書くと、近い体験ができる。こちらは書こうとすると、「あんたの続きはこれでしょ」という感じで文章が提案される。コードの場合はこれが便利で気持ちいいが、文章だと「違う」と思ってしまう。
頑固オヤジのように、自分でやりたいと思ってしまう場面は、対AI以外にもありそうだ。この時の感情は何なんだろうか？自分の方が上手くできると思っている？所有欲？自己顕示欲？  
今年の芥川賞を受賞した九段理江さんは ChatGPT を使いながら書いているらしい。どうして文章のプロが使っていて、素人の自分が使わないのか。  
つくるプロセスは大切で、その過程が一番楽しかったりもする。AIを使うと楽しさが楽しさが奪われると思っているのだろうか。エディターに助けを求めていたのに、いつの間にか自分の欲の話になってしまった。  
凝り固まった思考や欲は投げ捨てて、軽やかに生きたいなぁ。楽しくAIと生きる方法を考えたい。
